A Linguagem HTML

HTML (abreviação para expansão inglesa HyperText Markup, que significa Linguagem de Marcação de HiperTexto) é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.)

Até este momento, vimos os seguintes recursos na linguagem:

Recursos Tags associadas Descrição
Links <a> Criar Links e Páginas
Listas <ul> e <li> Criar listas e itens ordenados
<ol> e <li> Criar listas de itens não ordenados
Imagens <img> Inserir imagens
